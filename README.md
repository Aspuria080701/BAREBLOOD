# BAREBLOOD
Gothic maximalist linux theme for KDE plasma 6.5

- Custom icon pack
- Custom cursor pack
- Custom window decorations
- Custom background frame (Wallpaper plugin)
- Custom GTK theme
- Custom KDE color scheme
- Custom login screen (SLiM)

*WARNING*

All the art used to make this rice is *AI GENERATED* (except for the wallpapers)
