if you maximize a window you'll see the titlebar background off place probably..
that's because the background is relative to the size of the window when is maximized.
that can be altered by the screen resolution and bottom panels setted as "Always visible".

so you'll have to change the height value of the rect in the line 23 inside the decoration.svg file.
there is a formula you can use to calculate that value in relation to your resolution height.
if you have fixed bottom panels. just substract the panel height to the resolution height and that's h.

y=-(3460/53)+(175/212)*h
